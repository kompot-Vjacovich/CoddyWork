Типы данных (виды переменных)

  1. Числа - 9
  2. Строки - "строка"
  3. Массивы - [9, "строка", true]
  4. Объекты - {'свойство1': true, 'свойство2': 0, 'свойство3': "строка"}
  5. Булевый или логический - true или false

Создание переменных

  1. С помощью let - существует только между двумя ближайшими фигурными скобками.
  2. С помощью var - существует внутри функции или внутри всего кода.
  3. С помощью const - работает также как и let, только такую переменную нельзя изменить.

Что можно делать с переменными

  1. Числа - складывать, умножать, делить, вычитать.
  2. Строки -
    1. Складывать(склеивать) - "строка1" + "строка2".
    2. Изменять регистр - "СТРОКА".toLowerCase() => "строка", "строка".toUpperCase => "СТРОКА".
    3. Выбрать какой-то элемент из строки - let a = "Строка" -> a[2] => р
  3. Массивы -
    1. Складывать - massiv1.concat(massiv2).
    2. Добавлять - massiv.unshift() - вставить вперёд, massiv.push() - вставить в конец.
    3. Удалять - massiv.shift() - удалить первый элемент, massiv.pop() - удалить последний элемент.
    4. Выбрать какой-то элемент из массива - let a = ["1", "два", "3"] -> a[0] => "1"; a.indexOf("два") => 1.
    5. Сделать из массива строку - a.join("/") => "1/два/3".
  4. Объекты - зависит от значений свойств этого объекта.
  5. Булевые переменные - складывать, вычитать, умножать, делить(тогда false = 0, а
    6. true = 1).
