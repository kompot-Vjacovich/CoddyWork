Javascript
Название функции(метода) Описание
alert('Привет') Выводит всплывающее сообщение с текстом "Привет" и одной кнопкой.
prompt('Привет') Выводит всплывающее сообщение с текстом "Привет", с полем для ввода ответа и двумя кнопками. Чтобы узнать ответ можно сохранить его в переменную : let otvet = prompt('Вопрос').
confirm('Привет') Выводит всплывающее сообщение с текстом "Привет" и двумя кнопками. Чтобы узнать ответ можно сохранить его в переменную : let otvet = confirm('Вопрос'). Ответ может быть true или false
console.log('Привет') Выводит в консоль сообщение с текстом 'Привет'
